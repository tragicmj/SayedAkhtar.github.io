METEO.CAT

Web prototype for meteorological portal based on the work done for the Servei Meteorològic de Catalunya . Desktop version (1180+ pixels). The sample includes design (Figma, Adobe Illustrator) and web development (HTML5, CSS3, JavaScript). Weather display made with Carto technology (CartoCSS, CartoJS). Download map of the website structure. Work done for the public company Meteocat: proposed redesign of the version for desktop and web layout of sections of the version for mobile devices accessible from m.meteo.cat.
